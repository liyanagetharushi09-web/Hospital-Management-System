The Hospital Management System is a web-based platform built with Oracle APEX and Oracle Database 19c to streamline hospital operations. It provides a centralized solution for managing patients, doctors, appointments, prescriptions, pharmacy inventory, surgeries, billing, and staff records.

The system leverages Oracle SQL for data definition and manipulation, with PL/SQL triggers and procedures handling business logic such as automated ID generation, billing workflows, and appointment scheduling.

By normalizing the database into eight core tables — PATIENT, DOCTOR, APPOINTMENT, PHARMACY, BILLING, SURGERY, NURSE, STAFF — the design ensures data consistency, efficient queries, and scalability.

Key highlights include:
Unified billing system supporting multiple payment methods (Cash, Card, Insurance)
Flexible appointment booking across online, physical, and call channels
Surgery tracking with operating room assignments and outcome recording
Pharmacy management with stock levels, pricing, and expiry monitoring
Nurse–Doctor assignment for clinical supervision, plus separate staff records for non-clinical roles

This project demonstrates practical application of database design principles, entity-relationship modeling, and Oracle APEX integration, making it a complete hospital operations management solution.
